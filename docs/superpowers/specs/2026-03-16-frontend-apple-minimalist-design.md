# 前端 Apple 极简风格改造设计文档

**日期**: 2026-03-16
**状态**: 待审核

---

## 概述

将现有微信风格聊天界面改造为 Apple 官网极简风格，核心特征：极简留白、纯色背景、呼吸感布局、苹果蓝强调色。

---

## 设计决策

| 维度 | 决定 |
|------|------|
| 整体风格 | Apple 官网极简风 |
| 侧边栏 | 抽屉式，默认隐藏 |
| 消息样式 | 卡片式，白色背景，头像区分 |
| 主色调 | 苹果蓝 #007AFF |

---

## 1. 整体布局

```
┌─────────────────────────────────────────────┐
│  ☰  小雪                              ⊙    │  ← 极简顶栏 48px
├─────────────────────────────────────────────┤
│                                             │
│         ┌─────────────────────────┐         │
│         │  卡片式消息内容          │         │  ← 消息居中，最大宽480px
│         └─────────────────────────┘         │
│                                             │
├─────────────────────────────────────────────┤
│     [ 表情 ]  [ 输入框................ ] [发送] │  ← 底部输入区
└─────────────────────────────────────────────┘
```

- 背景：纯白 `#FFFFFF`
- 侧边栏：抽屉式，从左侧滑出，宽度 280px，带毛玻璃遮罩
- 无多余分割线，用留白分隔区域

```
┌──────────────┬──────────────────────────────┐
│              │                              │
│  小雪        │       （遮罩层）              │

│              │                              │

│              │                              │
│  ○ 小雪      │                              │
│  ○ 小明      │                              │
│              │                              │
│  ──────────  │                              │
│              │                              │
│  [创建角色]  │                              │
│              │                              │
└──────────────┴──────────────────────────────┘
     280px              毛玻璃遮罩
```

### 抽屉面板

| 属性 | 值 |
|------|-----|
| 宽度 | 280px |
| 背景 | `rgba(255,255,255,0.95)` |
| 滑入动画 | 300ms ease-out |
| 右侧阴影 | `0 0 20px rgba(0,0,0,0.1)` |

### 遮罩层

| 属性 | 值 |
|------|-----|
| 背景 | `rgba(0,0,0,0.3)` |
| 行为 | 点击关闭抽屉 |

### 抽屉内容

- 顶部：当前选中角色头像（60px）+ 名称 + 描述
- 分割线：1px `#E5E5E5`
- 角色列表：每项高度 56px，左侧头像 + 名称
- 选中态：背景 `#F5F5F7`，左侧 3px 蓝色指示条
- 底部：创建新角色按钮（胶囊形，边框样式）
- 记忆/画像标签：分段控件切换

## 13. 状态设计

### 发送中状态

- 输入框禁用
- 发送按钮显示加载动画（白色圆点旋转）
- 消息卡片添加半透明遮罩 + 中央加载指示器

### 加载历史状态

- 消息列表顶部显示加载指示器
- 使用灰色圆点动画

### 错误状态

- 发送失败：消息卡片右侧显示红色感叹号图标
- 点击感叹号可重试
- 网络错误：输入框上方显示红色提示条

## 16. 验收标准

- [ ] 页面背景纯白，无多余装饰
- [ ] 顶栏简洁，仅显示菜单图标、角色名、清空按钮
- [ ] 消息以卡片形式居中显示，最大宽度 480px
- [ ] 抽屉从左侧滑出，带遮罩层
- [ ] 输入框为胶囊形，发送按钮为苹果蓝
- [ ] 所有颜色符合设计规范
- [ ] 动画流畅自然
